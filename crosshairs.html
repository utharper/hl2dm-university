Crosshairs

Custom crosshairs from the community. Right click and select "Save Link/Target As..." to save each to your computer.

Installation and Removal

The crosshairs in HL2DM are stored in a file called halflife2.ttf. To install custom crosshairs, simply replace this file with the new one. To return to the default crosshair, simply delete this file and the game will automatically create the default one again.

This file can be found in the following location:

C:\Program Files\Steam\SteamApps\your_account_name\half-life 2 deathmatch\hl2mp\resource\
