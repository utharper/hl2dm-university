for it not to be a spam - advice to moviemakers from me: if you want you slo-mo to be smooth and with sound, shoot it slowed and make avi with no sound, then, using game engine, shoot it with normal speed and take only sound. Use progs like Nero Wave Editor to change sound length and pitch. Then u can synchornize sound and video using your movie editor
