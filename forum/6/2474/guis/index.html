0nti wrote:sven, first time I hear that name D; Sounds cool though ^_^
I don't know how to change the font, never done it myself, but you could try contacting those who have (mainly euros as far as I've seen) and asking :]
Yeah it's a scandinavian name but used pretty much in belgium

Ok open your clientscheme.res then search for this:

HudNumbers
{
"1"
{
"name" "HalfLife2"
"tall" "32"
"weight" "0"
"antialias" "1"
"additive" "1"
"custom" "1"
}
}

change the Halflife2 in the font you want to have (note: it has to be a .ttf file !)

If you want to use custom fonts then get a font and place it in your resource folder.
After doing that go to this:

//////////////////////// CUSTOM FONT FILES /////////////////////////////
//
// specifies all the custom (non-system) font files that need to be loaded to service the above described fonts
CustomFontFiles
{
"1" "resource/HALFLIFE2.ttf"
"2" "resource/HL2crosshairs.ttf"
}

Now just add the link to your custom font will look like this:

{
"1" "resource/HALFLIFE2.ttf"
"2" "resource/HL2crosshairs.ttf"
"3" "resource/YourFont.ttf"
"4" "resource/YourFont2.ttf"
}

Normally this should work, hope it helps you
