For about 2 months I couldnt launch hl2 without lagging hard time at launch, and once i'd join a game.

Same for TF2.

I'd also get massive slowdowns ingame.

I finally figured it out, CUSTOM SPRAY.

By hitting the console key by mistake I was horrified by what i saw.

Enough '' Failed to find vertex transparencies for conflictcustomspray.vmt'' in red text to fill out a .txt file up to 20 mb.

The game kept trying to find the godamn transparency layer or idk (don't ask me it doesn't make any sense)

I had sound looping for 30-40 seconds when joining a game and massive slowdowns in server list during refresh due to the game scanning my sray file endlessly. Removed spray and it works like a charm now D:
