Hi there,

For the last 3 months or so I have been secretly working on a map designed for AG2 gameplay. The result of which is a map which is sort of a remake of sadists pit but with new ideas. The central core theme is still the main part of the new map as it was with sadists pit. But now it is using a different set of textures which come from the same palette, and a theme of industry.

The map is of medium size, and is really just a main central room spanning 3 floors and an L shaped room running around the central room, and a few rooms, corridors and stairs running of it. Very simple in layout. High ceilings and lots of wide space to perform bunny hops and gauss jumps.

The map also contains some wirring machine sounds and a new lighting feature which produces a spinning fan shadow. Weapons are contained in little alcoves which keeps the map tidy.

The map is now available as part of AG2 - Download here
