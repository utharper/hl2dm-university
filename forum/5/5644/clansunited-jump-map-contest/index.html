Clans-United Jump Map Contest

Maxtasy

2011-02-17 21:48:45



Hey hey guys!


As you can figure from the headline, Clans-United is going to start a
Mapping-Contest especially for Jump maps.

If you like to play on jump maps, are creative and know how to use the
Hammer Editor then you might just be the perfect guy we are looking for.


The current jump maps around are kinda easy now and a bit boring so we are looking
for something that makes you feel really good when you manage the last level.
If you have any ideas how a map should look then show us and get started



These are the rules!
  • -Submission Deadline: 28th April.
    -Everybody is allowed to post their map from 21th to 28th April.
    -Posting maps before this date isnt allowed, or else the map will be disqualified!
    -No remakes are allowed.
    -The map has to be completely new, posting previously created maps isnt allowed.
    -Publishing the map before the 21. of April isnt allowed.
    -The map should be finishable, otherwise it would make no sense.
    -Intergrate the Clans-United logo into ur map


This criteria should be respected:
  • -Asthetics!
    -Complexity factor (more difficult is better coz the acutuall jumpmaps are very easy).
    -Frames per second.
The judges will be the community. A poll will be set up from 28th April.
The one with the most votes gets the good reputation of being the best jump mapper around and a place in the hall of fame on the CU site.
