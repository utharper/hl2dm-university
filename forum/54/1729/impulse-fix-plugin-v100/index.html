Impulse Fix Plugin - v1.00

Keeper

2008-05-13 23:06:03

Purpose:
  • This plugin was created to block the use of an impulse exploit. This plugin will simply deny the effects of the impulse and kick the offender.
Installation:
  • Extract the contents of this archive to a known directory.
  • VSP - standard plugin
    1. Ignore or delete the metamod and keeper directories.
    2. Place the keeper_noimpulse_hl2.vdf file in the addons directory of the server ( if there is no addons folder, create it under the main hl2mp folder ).
    3. Place the keeper_noimpulse_hl2.dll file in the addons directory if you have a Windows server, or the keeper_noimpulse_hl2_i486.so file if you have a Linux server.
    4. Restart the server.
    5. Type plugin_print to see if you should see
      Keeper's NoImpulse Plugin (VSP) - v1.00 - http://www.hl2dm-university.com
    Source MetaMod ( SMM ) plugin
    1. Ignore or delete the keeper_noimpulse_hl2.vdf, keeper_noimpulse_hl2.dll, keeper_noimpulse_hl2_i486.so files in the main directory.
    2. Place the keeper_noimpulse_hl2_mm.vdf file in the addons/metamod directory of the server.
    3. Place the keeper_noimpulse_hl2_mm.dll file in the addons/keeper/bin directory if you have a Windows server, or the keeper_noimpulse_hl2_mm_i486.so file if you have a Linux server.
    4. Restart the server.
    5. Type meta list to see if you should see
      Keeper's NoImpulse v1.00 Keeper RUN
Notes:
  • If you are using SourceMM, please update to the latest version: 1.4.3
    If you are not already using SourceMM, I would suggest just using the VSP version.
