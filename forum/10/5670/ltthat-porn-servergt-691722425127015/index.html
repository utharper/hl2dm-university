Installing the OB mod:

Step 1: Install a clean SRCDS with orangebox and hl2mp using the following commands:

steam -command update -game hl2mp -dir

steam -command update -game orangebox -dir

Step 2: Download and unzip http://www.mediafire.com/?53x52vm5sa35kga
Into the orangebox directory. The orangebox directory should now contain both the hl2mp folder and the newly extracted hl2mpob folder.

Step 3: Edit the file “YOUR.SERVER.IP_PORT/orangebox/hl2mpob/cfg/server.cfg”
by adding the line "sv_password abc123" in order to introduce a password, so that the server does not violate the private server criterion.
It does not matter if the server is paid for as public; this criterion exists within the code of the mod.
Any public server can have the password removed entirely after the initial boot-up.

Step 4: Modify the srcds_run command line so that where it once said
"-game hl2mp" so it now says "-game hl2mpob".

Step 5 (conditional): Linux servers will need legacy .dll files in the bin folder: to download = http://www.mygamecp.com/Uploads/Support ... ngebox.rar
Windows servers "shouldn't" need them, and I don't think mine did.

Step 6: Start server through web interface.
