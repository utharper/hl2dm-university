your mic may be muted in the control panel, check to see if it is or not.
Go to control panel, select sounds and audio devices, on the first tab under device volume hit the advanced tab, and look for the microphone box to see if the mute option is checked, if it is uncheck it and retest. If you dont see a microphone box on the upper left corner select options and check show advanced.

It could be a setting in your sound drivers.
Also check on Teamspeak that the input and outputs are the same device.

EDIT:

Or maybe after fiddling about.
Go to the windows sound panel select "options>>properties>>click recording
You should see a window like this

Now check the "what you hear" box click OK.

After clicking OK Should see something like this

Now check the MICROPHONE select box it should uncheck "what you hear" automatically.
When done return to "playback"

Easy way to test is use Tamspeak connect to an empty server and start playing some tunes if the green light doesn't come on then it is problem solved.
