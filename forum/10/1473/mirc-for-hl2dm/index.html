Well i wanted to make a guide for mirc so here goes,

download the small program @ http://www.mirc.com or if you don't like clicking to the d/l tab -> http://mirc.com/get.html

after you install it, mirc will say you have 30 days to regisiter, well really you have 30 days to not have to wait 2 seconds b4 you open mIRC so your fine.
Once you open mirc it should bring you to the Options tab:



You can find thins under Tool>options or press alt+o

set your name e-mail etc

Now goto the Server tab under connect

Hit add and this window should pop up



Name it whatever you want my is nino

IRC server: irc.us.gamesurge.net
Port : 6667

everything else blank > hit ok
Now highlight that new server name you made and hit select
This will take you back to Connect. Press 'Connect To server'

K now you will be connecting to the gamesurge network, now to join the Channel

Click Favorites from the main menu.
Click Organize Favorites ( you can go and delete these random channels at some point)




Now add #hl2dm

under channel enter #hl2dm
no pasword
no Description
All networks
Check the Box 'Join on connect'
hit add
hit ok

Now you are done. you can close mirc and now all you have to do is launch the program and connect. And you will Auto-Join #hl2dm.
