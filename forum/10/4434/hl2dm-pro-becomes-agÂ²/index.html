Pernicious wrote:"removing hand grenades before starting a round"

Hi all,

First of all, please note that I don't want to maintain any polemic. You have your opinion that I respect and I have no concern about that. It 's your liberty, it 's our liberty, let's preserve it.

Now I just would like to come back to this post about handgrenade which generates a big amount of posts but on a misunderstanding. I think that this sentence "removing hand grenades before starting a round" is coming from our readme.txt related to version 1.9.3. It has nothing to see with spawning behaviour. In fact, when playing Last Team Standing or Arena game mod, it was reported that when a round was finished, if an alive player was launching a handgrenade to a spawnpoint 2 seconds before starting the next round, you could kill a player which was spawning in this spawnpoint. It was very annoying and the community asked us to prevent it. So I decided to remove all hand grenades in the battlefield (not the one you have when you spawn) before starting a new round.

Last thing, you must all understand that in our mod, you have 2 standard gameplay : one is the regular hl2dm gameplay (the one you know when playing vanilla, it has exactly the same behaviour, same weapons when spawning, same quantity of ammo...) and the other is called pro and is including different feature (gauss, bunny hopping...). But there is also a built-in "gameplay sdk" if you want to create your own gameplay by choosing weapons you want to allow, weapons you want on spawn, damage per weapon, ammo per weapon, push impact, speed of players, speed of missile, speed of bolt...many many parameters. There can have many gameplay in additional to the 2 basic ones.

I hope it clarifies.

J.

As already said, AG2 is based on the source HL2DM code meaning that it is 100% compliant with vanilla map but in fact with many more maps as it is also compliant with maps coming from many other mods like css, hidden... As AG2 is under OB engine, it is also able to load maps from TF2 and all OB engine based mods like DODS...
According to editing maps, you must understand that it is not mandatory. There are thousands of dm maps, we can't edit them all. Due to this fact, we develop an algorythm which is dynamically analyzing the map to add specific item in the map when needed. For example, for "pro" gameplay we are adding gauss gun and long jump which are key items in this gameplay (of course, it won't be visible in "vanilla" gameplay). In the same idea, if you want to plat CTF in lockdown or any other maps which does not include any flag zone, it will analyze the map to add these flags in the "best" location for both teams. No need to edit maps.

We tried to make it as easier as possible to prevent any players from having to use their brain too much on anything else than fragging
