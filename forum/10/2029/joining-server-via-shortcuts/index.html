How do you do it so that your in-game menu has where you can click and it sends you into a specific server you have?

keefy

2008-07-04 21:06:44

You need a file called gamemenu.res once got it place it in C:\Program Files\Steam\steamapps\USERNAME\half-life 2 deathmatch\hl2mp\resource

You can copy and paste the following into a text file, just change the server IP accordingly and their Labels or add more.
Save as gamemenu.res

Code: Select all

"GameMenu"

{
"1"
{
"label" "#GameUI_GameMenu_ResumeGame"
"command" "ResumeGame"
"OnlyInGame" "1"
}
"2"
{
"label" "#GameUI_GameMenu_Disconnect"
"command" "Disconnect"
"OnlyInGame" "1"
}
"3"
{
"label" "JOIN Castle Roman Sanctuary only"
"command" "engine connect 193.47.83.200:27016"
}
"4"
{
"label" "JOIN Sanctuary map cycle"
"command" "engine connect 193.47.83.200:27015"
}
"5"
{
"label" "JOIN Keefys server"
"command" "engine connect 81.19.219.24:27025"
}
"6"
{
"label" "#GameUI_GameMenu_FindServers"
"command" "OpenServerBrowser"
}
"7"
{
"label" "#GameUI_GameMenu_CreateServer"
"command" "OpenCreateMultiplayerGameDialog"
}
"8"
{
"label" "#GameUI_GameMenu_Options"
"command" "OpenOptionsDialog"
}
"9"
{
"label" "#GameUI_GameMenu_quti"
"command" "quti"
}
}
