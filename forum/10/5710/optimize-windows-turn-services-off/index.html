The Argumentalizer

2011-04-16 05:00:24

I have been interested in a fast system for years and have uncovered many tricks to secure up and speed up Windows.
I am posting a services log to show folks what can be disabled.

WARNING! - This is a single PC not part of a network. If you have a network, all bets are off.
Also, Some components in Windows will not work if their service is set to manual. Remote Procedure Call (a Huge deal) HAS to be set to automatic.
Also, be aware, i do not use search or restore. Worms can use restore to survive. It hasn't ever really worked and is a trememndous resource hog.
So, i disable Restore in the System.

This is for XP. I am not as familiar with 7 or Vista, but i am sure many of the same services run.
And know this: Microsoft has their OSs set to install with an incredible array of useless resource abusing services, so you can do everything.
If you are a gamer with a single machine and want it as responsive as possible, the first step is:

Disable Indexing and Search
Disable System restore
Disable all graphic nonsense except maybe Themes, Smooth edges of fonts (ClearType anti-aliasing) and Show contents while dragging.
Disable dozens of Networking services you don't need, that will run in memory.
Here is the list. If you are unsure about a service, set it to MANUAL, meaning the service will start only if needed. Automatic means the service is running.
The ones disabled shouldn't cause you any trouble
