I just finished a build with that exact board.

One of the issues I had was that setup wasn't recognizing the SATA drive, even though an SP2 disk is supposed to. I had to fool with a workaround, you may have to as well.

I would suggest using the 2000 disk just to format the drive ( I usually give the OS its own partition ) and then exit to retry the XP cd. Try to get the SATA drivers from the Gigabyte disk onto a floppy so you press f6 when it asks for RAID or 3rd party drivers. Also enable USB keyboard support in the bios as soon as you can, I had to use a ps/2 keyboard just to access the bios the first time.

glhf...once you get it going I think youll be happy with the board...although the dual lan ports are useless for gaming...

in teh right pane, right-click in an open space and select New String Value
