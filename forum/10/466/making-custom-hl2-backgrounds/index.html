Making Custom HL2 backgrounds



What you're gonna need.
1) Photoshop (or similar editting software) and this TGA Plugin
2) Vtex converter included in steam
3) Time on your hands

firstly prepare your image

id suggest using an image that is equal to your current hl2 screen resolution.
it makes things easier and looks better then streatching a smaller image.

ok, save your image as tagra file format (.tga) and call it background01.tga

before you proceed you must ensure you image measures to the power of 2,
so if you resolution was 1280 x 1024, select image size, unckeck "constrain
image size" and change the size to 1280 x 1280. it will look out of shape but
the console will sort this out, trust me

Ok
place the background01.tga file here,

program files\valve\steam\steamapps\yournamehere\half-life 2 deathmatch\hl2mp\materialsrc

Now you need to convert the tga into a vtf format using the vtex.exe

open this directory.
program files\valve\ssteam\steamapps\yournamehere\sourcesdk\bin

and drag the background01.tga from materialsrc onto vtex.exe


The file background01.vtf will now be created in

program files\valve\ssteam\steamapps\yournamehere\half-life 2 deathmatch\hl2mp\materials


simply create a folder here, called console and place the background01.vtf inside.

Load up and there it is.


easy!!

any problems PM / email. if your stuck i dont mind making it for ya.

Pig Out!!

Or you do it with VTFEdit. Costs 0$ and seems much easier to me.
http://nemesis.thewavelength.net/index.php?c=178

Start VTFEdit, click on the white Page symbol that says “Import”, browse to your desired picture or wallpaper and open it.
Save it with the standard values, important thing is to have Resizing Method “Nearest Power of 2” and equal values for Height and Width.
Name the picture “background01.vtf” and save it into “hl2mp/materials/console/" folder.
Done.
