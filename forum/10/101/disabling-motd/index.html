The motd is loaded and saved in the network string table. This means that all the clients sync up with this string table. The client then takes the portion that is MOTD and displays it itself. So the displaying of the MOTD is purely client ended. The file/contents of the MOTD is server ended. This is why there's a cap on the size of the MOTD.
