BlackRaven wrote:i mean what if i have an image id like to use as the menu image?
in PS, grab any image you want to use and stretch it over a 1024 * 1024. save as vtf (u might need photoshop plugin for that)
replace and rename with files.
Leave VMT files as they are - unchanged.
Done.

BTW: using the PS VTF plugins eliminates the need for saving as TGA as in popper's tut. simply save as...vtf your image and you'll be good.
