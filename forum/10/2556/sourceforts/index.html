Sourceforts is a mod of hl2dm and allows players build their own forts using blocks. Now, the best part is after the build phase, the combat phase begins and the game has been made into a class based version of hl2dm basically. It's super fun and the competitive games there are amazing and intense. The best part about it probably is that trick jumping is actually really useful as you can see in this video. http://www.youtube.com/watch?v=wZWn4_azUFU. There are jumps with the rocket, slams, and nades and the game is just really interesting in almost all aspects.

You can download it at www.sourcefortsmod.com. Have fun
