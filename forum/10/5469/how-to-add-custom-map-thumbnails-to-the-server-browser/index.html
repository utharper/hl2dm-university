I have created a few rars that offer the custom map thumbnails for the maps I currently have on our CGC dm server, Ground Zero. These are the pictures you see next to a server in the server browser. As you may know, custom maps show a big question mark. This fixes that.
If you dont feel like downloading my rar files from this thread, you may enter the server (66.225.194.156:27015) and it'll send them to you. The one for fastdl folders is labeled as such and there are readme files in both rars explaining how to create these yourself and also how a server owner can add them to their server so players can download them from it. Anywho, anyone that feels like adding more, feel free to add them to this thread.

Here are the maps I did. These work for all servers running these particular maps!! Once you have these, they will show up in your server browser if they are the map being played.
