Newversion 1.82 out on friday with many updates and fixes.
http://www.hl2dmpro.com/
Improvements included in this version:
. Added : advanced hit registration routine making it more efficient (it can be deactivated by admin or by setting cvar pg_sv_enhancedhireg to 0). Please note that it superseeds algo from Valve.
. Added : new trigger_pushinstant for instant jump pads
. Added : ability for an admin to transform trigger_push to instant trigger_push so that trigger_push acts like trigger_pushinstant
. Added : cvar pg_sv_ctf_roundbased to play CTF game mods with or without rounds
. Added : cvar pg_sv_grabfloatingweapons to allow grabbing floating weapons using gravity gun
. Added : cvar pg_sv_grabfloatingitems to allow grabbing floating items using gravity gun
. Added : if a player quits during a match, his score remains displayed in scoreboard
. Added : cvar pg_sv_reg_fov and pg_sv_reg_fov_admin so that admins can prevent modifying FOV in "Original HL2MP"
. Changed : maps pg_sacred, pg_roman_sanctuary, pg_the_longest_day, pg_fragging_yard, pg_deck17, pg_campgrounds and pg_stadium are now using trigger_pushinstant
. Changed : spawnpoints reviewd in pg_stadium
. Changed : gauss gun attack1 is now a straight laser
. Changed : in Football gamemod you can only push the ball carrier using gravity gun
. Fixed : in Football gamemod, you can push ball carrier in "Original HL2MP" gameplay
. Fixed : bug with blue flag
. Fixed : a user with ID_PENDING could be able to join a match without having been allowed
. Fixed : IRC bug related to private message
. Fixed : stucked while trying to do a long jump when you 're unducking
. Fixed : item_healthvial and func_healthcharger are no more allowed in LTS/Arena/Tourney
. Fixed : RPG missile could not be guided even with red dot sometimes yeehaaa
. Fixed : Long Jump is now allowed while unducking
. Fixed : no more flickering animation while RPG is reloading after switching from a weapon to it
. Fixed : textures/models error when using "Server Settings" capability
. Fixed : possible server crash when players were leaving during a tourney in gamemod Tourney
. Fixed : possible war with only one player when a player quti in gamemod Tourney
. Fixed : when moving very fast and holding orb, it can kill launcher (if allowed) just after launching it
