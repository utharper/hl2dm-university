No but a better vid card will. A better HD will let the game run a bit smoother. Its always better to run the game on an internal HD as opposed to an external one because communication between the HD and the CPU is faster and more efficient. A 10,000 RPM drive will run faster and theoretically make the game run smoother and faster but Im not sure how much it is actually noticable. Also a HD with larger cache size will run faster.
