Ping spikes during wireless gaming can be sorted by turning off the Wireless Zero Configuration service. Put this in a text file:

net stop wzcsvc

and rename it stopwzc.bat. Run that and it will stop the sertvice regularly polling your wireless card (which causes the ping spikes). If you need to restart the service make another file and change stop to start. Or you can just run it from the run option in your start menu. Not a total solution, but makes Wifi gaming a better experience.
