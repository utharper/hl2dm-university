Here's a tip to speed-up your Firefox (any version and OS).

Open 'about:config'

Set “network.http.pipelining” to “true”

Set “network.http.proxy.pipelining” a “true”

Set “network.http.pipelining.maxrequests” to a value of about 30. This will increase the request that Firefox makes to a site, be aware of your bandwith for other computers using the same connection.

Right click where ever you want to and choose “New -> Integer”. Name it “nglayout.initialpaint.delay” and then set it “0”.
