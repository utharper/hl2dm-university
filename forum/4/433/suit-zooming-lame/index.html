Ko-Tao

sv_allow_wait_command 0

Disables scripting. This setting is active on most competitive servers and in the cal.cfg.

If youre playing on a server that has the above command set to 1, the server op has decided that scripting is acceptable, so instant zoom and the like should be expected.

Btw, valve condones scripting- specifically, note this line from the rc file in your config folder:

// run a user script file if present
exec autoexec.cfg
