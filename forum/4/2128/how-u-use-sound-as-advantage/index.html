yeah doing it in game with someone is the best way to learn how to use it. You pretty much have to work on each map you plan to use it on. Have a friend go to each pick up while you stand in different areas, have him tell you where he is at and then listen for the pickup, get to know where it sounds like it is and how strong or how faint the sound is from where you are standing. Takes some practice but you will start to recognize it. Lockdown is one of the better maps to try to learn this on, but if you don't play LD much it wont help you on other maps.
