Heres my setup:

Hud and Crosshair Color:
"Normal" "0 255 0 255"

Zoom Crosshair Color:
"ZoomReticleColor" "0 0 0 0"

Chat Text Color:
Chat.TypingText "Orange"

So, a bright green crosshair, 100% opaque (this controls most of the hud too), a faded colorless outline for the zoom reticle (cant remove it completely) and the stock orange chat text. Im not sure where it defines the colors for team red/blue chat texts that come up onscreen, but presumably those can be edited too.

I also have my crosshair set to non-additive (crosshair font | additive 0) and ingame hover-over enemy text size set smaller than stock (TargetID font | tall 16).

Lastly, if someone knows the clientscheme line or other .res file that controls the translucent black spec bars, please do tell.
